  • Nainštalujte SvelteKit
  • Nainštalujte @sveltejs/adapter-static and gh-pages. `npm i -D @sveltejs/adapter-static gh-pages`
  • Aktualizujte súbor svelte.config.js zmenou adaptéra-auto na adaptér-statický. `import adapter from '@sveltejs/adapter-static';`
  • Pridajte nasledujúce do súboru `src/routes/+layout.js` `export const prerender = true;`
  • Pridajte skript do súboru package.json. `"gh-pages": "npm run build && npx gh-pages -d build"`

Teraz stačí spustiť `npm run gh-pages`.
